مشروع الاسبوع :

إنشاء موقع الكتروني عبارة عن حاسبة بلوت كمهمة رئيسية وعرض قوانين وطرق اللعبة كمهمة فرعية.

المتطلبات :
- حساب الصكة ودور الموزع.
- عرض قوانين اللعبة دون الاخلال بالمهمة الاولى.

تجهيز الواجهة :
- إنشاء صفحة تعريفية لي, تكون عند دخول الموقع وبعد نهاية كل صكة.
- إنشاء صفحة الحاسبة, وتتضمن :
    - مكان لاسمي.
    - مكان سهم الموزع.
    - مربعين ادخال النتيجة.
    - زر حساب النتيجة المدخلة.
    - جدول نتائج اللفات.
    - مربعين عرض نتيجة الفريقين.
    - زر التراجع وزر اعادة الصكة وزر القوانين.
- إنشاء صفحة القوانين, وتتضمن :
    - اقسام القوانين وعند الضغط على كل قسم يظهر تعريف القسم.
